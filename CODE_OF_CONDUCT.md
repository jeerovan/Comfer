# Code of Conduct

## Our Pledge

We want this project to be a welcoming, respectful, and constructive space for everyone. Contributors, maintainers, and participants are expected to help create a community where people can collaborate without harassment or hostility.

## Our Standards

Examples of behavior that helps create a positive environment include:

- Being respectful and considerate in discussion
- Giving and receiving feedback constructively
- Assuming good intent while staying honest about problems
- Focusing on what is best for the project and its community
- Showing empathy toward other contributors and users

Examples of unacceptable behavior include:

- Harassment, insults, or personal attacks
- Discriminatory or exclusionary language or behavior
- Trolling, deliberate intimidation, or sustained disruption
- Publishing private information without permission
- Any conduct that would be inappropriate in a professional setting

## Scope

This Code of Conduct applies within project spaces and in public spaces when someone is representing this project or community.

## Enforcement

Project maintainers are responsible for clarifying and enforcing these standards. They may remove, edit, or reject comments, commits, code, issues, and other contributions that do not align with this Code of Conduct.

## Reporting

If you experience or witness unacceptable behavior, open a private discussion with the maintainers through GitHub so the situation can be reviewed. Reports will be handled as respectfully and confidentially as practical.
